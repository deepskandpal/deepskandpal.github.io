Books you can read for Data Science and ML

I’m more of a books guy than a video guy when it comes to using resources to learn something new. So naturually when I started my Data Science journey I was always looking out for books which i get my hands on for learning new stuff.

So here’s my list of books one can read when learning Data Science

Disclaimer: Only proceed if you are an avid reader or you will just end up hoarding books which will take unnecessary space in your closet or your disk drive

lets start with the basic and the obvious ones

For Beginners

  1. Hands on Machine Learning with Scikit-Learn, Keras and Tensorflow: Anyone who has even a remote interest in this field proabably already knows about this book. This is book which gets recommended to death by Anyone and Everyone and for good reason too. It
  2. Data Mining
  3. the 100 page machine learning book
  4. Deep learning in python

Now comes the interesting ones

For Intermediates

  1. Machine Learning
  2. Practical Natural processing
  3. Machine Learning engineering
  4. Practical Statistics for Data Scientists
  5. Interpretable machine learning

the next set books although is tagged as advanced they are not advanced in terms of the material they teach , its the depth they cover for each topic which makes them sort of a refernce manual for some really advanced use-cases

For Advanced users:

  1. Pattern recoginition and machine learning
  2. Deep Learning
  3. The elements of statistical learning
  4. The Data Science Handbook
  5. Causalilty
