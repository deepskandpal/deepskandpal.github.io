Introduction to Arrays

An array is a fundamental data structure used to store a collection of elements, typically of the same data type, in contiguous memory locations. Each element is identified by an index or a key.

Key Characteristics

  • Fixed Size (Static Arrays): In many languages, traditional arrays have a fixed size defined at creation.
  • Dynamic Size (Dynamic Arrays/Lists): Languages like Python offer dynamic arrays (lists) that can grow or shrink.
  • Contiguous Memory: Elements are stored next to each other, allowing for efficient index-based access.
  • O(1) Access: Accessing an element by its index is typically a constant time operation.
  • O(n) Insertion/Deletion (Worst Case): Inserting or deleting elements in the middle may require shifting subsequent elements.

Common Operations & Complexity

  • Access (by index): O(1)
  • Search (linear): O(n)
  • Insertion (at end, amortized for dynamic): O(1)
  • Insertion (at beginning/middle): O(n)
  • Deletion (at end): O(1)
  • Deletion (at beginning/middle): O(n)

Use Cases

  • Storing lists of items.
  • Implementing other data structures (stacks, queues).
  • Lookup tables (when used with indices).
