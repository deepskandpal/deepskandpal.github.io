    [[Binary trees and Binary search trees]] [[Recursion]]

    Problem

    Given the root of a binary tree, return its maximum depth.

    A binary tree’s maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

    ![[binary_tree_height.png]]

    Brain storming

    The height of a binary tree can be defined at each level by “the maximum height of any of my left and right child plus 1” . The core here is at each node ,

    1. getHeight from my left child tree
    2. getHeight from my right child tree
    3. Height at the node is max(left_height , right_height) + 1

    This is a post order traversal left , right root

    below is the python implementation

    
class BinaryTree:
	def __init__(self, val, left=None, right=None):
		self.val = val
		self.left = left
		self.right = right


def getHeight(root:BinaryTree):
	if root is None:
		return -1
	
	left_height = getHeight(root.left)
	right_height = getHeight(root.right)
	current_height = max(left_height, right_height) +1
	
	return current_height
