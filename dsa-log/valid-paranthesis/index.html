    [[Linked List Stacks and Queues a Primer]]

    Problem

    You are given a string containing just the characters (, ), {, }, [ and ]. Your task: Write a function is_well_formed(s) that determines if the input string is “well-formed” (also known as “valid parentheses”).

    A string is well-formed if:

    1. Every opening bracket has a corresponding closing bracket of the same type.
    2. The brackets are closed in the correct order.

    Examples:

    • "()[]{}" -> True
    • "([{}])" -> True (This shows the nesting)
    • "(]" -> False (Wrong type)
    • "([)]" -> False (Wrong order)
    • "[" -> False (Unclosed)
