    [[Linked List Stacks and Queues a Primer]]

    Problem

    You are given two linked lists, L1 and L2, and you are guaranteed that each list is already sorted in ascending order.

    Your task: Write a function merge_two_sorted_lists(L1, L2) that merges the two lists into a single, perfectly sorted linked list that contains all the nodes from L1 and L2. You should do this by rearranging the existing nodes, not creating new ones.

    Example:

    • L1 = 2 -> 5 -> 7 -> None

    • L2 = 3 -> 11 -> None

    Result: 2 -> 3 -> 5 -> 7 -> 11 -> None

    Brainstorming

    the intuitive idea is this. Imagine you have a deck of cards both which are sorted. How would you merge them. You would pick 1 card each from the top of the stacks. Then you will see which one is smaller and place that on a separate new stack. You will do this for each of the top cards until one of the stack will finally be empty. Then you can pick the remaining stack of cards and place them at the top of this new list

    The algorithm

    1. Starting point : We will build a new list using the dummy head pattern and a tail pointer that will do all the appending dummy_head = ListNode() tail = dummy_head
    2. We will iterate as long as we have cards in both stacks while L1 is not None and L2 is not None
    3. Compare and check inside the loop l1.data <= l2.data
      1. we attach it to the end of tail tail.next = l1 and then advance the l1 pointer
    4. else l2 is smaller so we then attach that at the tail end tail.next = l2 and then advance the l2 pointer
    5. This is CRITICAL after a node is attached we advance the tail pointer tail = tail.next
    6. After the while loop is completed then we can attach the remaining of the 2 list
      1. tail.next = l1 or l2
    7. Our sorted list is attached to the dummy head we just return that return dummy_head.next
    
class ListNode:
	def __init__(self, data=None, next_node=None):
		self.data = data
		self.next = next_node
		
		

# L1 = 2 -> 5 -> 7 
l1_node7 = ListNode(7) 
l1_node5 = ListNode(5, 
l1_node7) L1 = ListNode(2, l1_node5) 

# L2 = 3 -> 11 
l2_node11 = ListNode(11) 
L2 = ListNode(3, l2_node11)

def merge_two_sorted_list(l1:ListNode, l2:ListNode):
	dummy_head = ListNode()
	tail = dummy_head
	
	while l1 is not None and l2 is not None:
		if l1.data <= l2.data:
			tail.next = l1
			l1 = l1.next
		else:
			tail.next = l2
			l2 = l2.next
		tail = tail.next 
	
	tail.next = l1 or l2
	return dummy_head.next
