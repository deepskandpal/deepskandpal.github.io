    Problem

    Given a non-empty list of words, return the k most frequent words. The Twist: Your answer should be sorted by frequency from highest to lowest. If two words have the same frequency, the word with the lower alphabetical order comes first.

    Example:

    • Input: [“i”, “love”, “leetcode”, “i”, “love”, “coding”], k = 2
    • Frequencies: “i” (2), “love” (2), “leetcode” (1), “coding” (1).
    • The top 2 frequencies are “i” and “love”.
    • Since they have the same frequency, we sort them alphabetically: “i” comes before “love”.
    • Output: [“i”, “love”]

    Brainstorming

    We need to first compute the frequency of words and then make sure that we are able to get “TOP K” frequent words

    TOP K would mean we would need to use heaps and we want most frequent so we will use a max heap. But heaps cannot process 2 things at once therefore we would need to club the word and the frequency together , we can either use a hash map or we can use a tuple

    We also need to make sure that if 2 words have same frequency then the order should be alphabetical . Tuples do this automatically for us so if (-2,“i”) and (-2, “love”) are compared then the the first tuple will be given top priority.

    so for max heap we will keep (-frequency , word)

    we will first compute the frequency and then use heaps to keep track of top k

    from collections import Counter
import heapq

input = ["i", "love", "leetcode", "i", "love", "coding"]
frequencies = Counter(input)
max_heap = []
K = 2

for word, count in frequencies.items():
	if len(max_heap) < K:
		heapq.heappush(max_heap, (-count, word))
	else:
		if (-count, word) < max_heap[0]:
			heapq.heappushpop(max_heap,(-count, word))

result_list = []
while max_heap:
	result_list.append(heapq.heappop(max_heap)[1])

print(result_list[::-1])
