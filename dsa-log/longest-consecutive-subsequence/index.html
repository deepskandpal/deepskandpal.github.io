    The Problem:

    Longest Consecutive Sequence (LeetCode #128)

    Given an unsorted array of integers nums, return the length of the longest consecutive elements sequence.

    You must write an algorithm that runs in O(n) time.

    Example 1:

    • Input: nums = [100, 4, 200, 1, 3, 2]
    • Output: 4
    • Explanation: The longest consecutive elements sequence is [1, 2, 3, 4]. Its length is 4. Notice that these numbers are not next to each other in the input array.

    Example 2:

    • Input: nums = [0, 3, 7, 2, 5, 8, 4, 6, 0, 1]
    • Output: 9
    • Explanation: The sequence is [0, 1, 2, 3, 4, 5, 6, 7, 8]. Its length is 9.

    The 0(n log n) solution:

    1. you can sort the array
    2. We then define a current_length and max_length variables both set to 1
    3. Then for each element check if it is similar to the previous element , if they are then its a duplicate hence we skip any increment and checks for Consecutive subsequence
    4. if the current number and the previous number defer by 1 then we increment the current length by 1 and store the max of max_length and current length in max length
    5. if its not that means the sequence is broken hence we reset the current_length back to 1
    nums = [100, 4, 200, 1, 3, 2]

nums.sort()

max_length = 1
current_length = 1

for i in range(len(nums)):
	if nums[i] == nums[i-1]:
		continue
	elif nums[i] == nums[i-1] +1:
		current_length +=1
	else:
		max_length = max(max_length, current_length)
		current_length = 1

print(max(max_length, current_length))
