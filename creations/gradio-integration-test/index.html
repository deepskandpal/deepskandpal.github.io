    Gradio Integration Test

    This page demonstrates how we can embed interactive Gradio apps directly into our Hugo blog using Gradio Lite.

    Simple Text Classification Demo

    Below is a live, interactive sentiment analysis model running entirely in your browser:

    transformers_js_py from transformers_js import import_transformers_js import gradio as gr

    transformers = await import_transformers_js() pipeline = transformers.pipeline pipe = await pipeline('sentiment-analysis')

    async def analyze_sentiment(text): if not text.strip(): return 'Please enter some text to analyze!' result = await pipe(text) return f'Sentiment: {result[0]["label"]} (Confidence: {result[0]["score"]:.2f})'

    demo = gr.Interface( fn=analyze_sentiment, inputs=gr.Textbox(label='Enter text to analyze', placeholder='Type something here…'), outputs=gr.Textbox(label='Analysis Result'), title='Sentiment Analysis', description='This model analyzes the sentiment of your text using Transformers.js running in the browser!' )

    demo.launch()

    Integration Benefits

    ✅ Serverless: No backend infrastructure needed
    ✅ Privacy: All processing happens in your browser
    ✅ Fast: Low latency after initial load
    ✅ Interactive: Perfect for educational content
    ✅ Static Site Friendly: Works with GitHub Pages

    Technical Notes

    This demo loads a sentiment analysis model from Hugging Face and runs it entirely in your browser using:

    • Gradio Lite for the interface
    • Pyodide for Python runtime in browser
    • Transformers.js for the ML model

    Note: Initial load may take 10-15 seconds as it downloads the Python runtime and model.
